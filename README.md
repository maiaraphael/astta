# Astta

Site institucional da **Astta** — empresa de desenvolvimento de websites.

## Sobre
A Astta é especializada no desenvolvimento de sites modernos e responsivos para empresas e profissionais.

## Tecnologias
- HTML5
- CSS3
- JavaScript

## Como rodar localmente
Abra o arquivo `index.html` no navegador ou utilize uma extensão como Live Server no VS Code.
